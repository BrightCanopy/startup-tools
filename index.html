Startup Tools

This is a curated list of products and services specifically useful to startups and bootstrappers.

This is a replacement for the "Cool Tools" folder in my browser bookmarks. I'll continue to migrate things in. Eventually this should be a long list. What other information should we be tracking for each? I'm hoping with collaboration, I'll learn of many I've never heard of before.

There is some redundancy here because one size doesn't always fit all, and there are sure to be gaps. Please suggest new additions or changes, by opening an issue or creating a pull request.

Things I Can Recommend

I have either used, or know someone who has used each of the services with great results. (Any exceptions are noted inline).

  • Amazon Web Services - The swiss army knife of cloud services. Does just about everything, but may not be the best tool for any one particular job. Lambda is especially interesting for a startup since it is pure pay-per-use.
  • Clerky - Incorporation services with a very specific focus on startups which may eventually be VC funded.
  • Dillinger - Open Source editor for github markdown (great for MVPs)
  • DigitalOcean - Cloud hosting based on Docker droplets.
  • Google Apps - Collaborative documents, spreadsheets, email and more. Especially good for MVP surveys and even simple MVPs.
  • Groove - Streamlined desk software with a scalable pricing model.
  • Heroku - Cloud hosting (Ruby Java Node.js Python PHP Clojure Scala) with lifecycle management and many plug-ins.
  • Instapage - Freemium Landing Page Builder with full A/B testing capabilities and a wordpress plugin.
  • Invision - A very polished online design and wireframe tool with a free tier.
  • NearlyFreeSpeech.net - Low cost, git deployment friendly, pay-what-you-use hosting. Great I have used them for MVPs and know they handle some big production loads well.
  • Keen IO - Analytics backend as a service.
  • launch27 - Booking engine for local service businesses.
  • Prose IO - An editor for github markdown (useful for MVPs)
  • QuickMVP - Integrated landing page and Google Ads for rapid idea validation.
  • Slack - Team messaging, archiving and search with an automation api and bots.
  • Stripe - Startup friendly payment processing with a powerful API.
  • Zapier - Quick and easy API integration for stitching together MVPs or even production services.

Things I have not used but which look useful

  • SalesIntelligent - Sales prospecting automation software. Useful for finding contacts within companies.
  • Survata - Online surveys sent through their own network of content publishers. Useful to validate an idea.
  • Zenefits - Health Insurance, Payroll, and other HR services.
